# Linux Command  

~ 用户目录
/ 根目录
ls 列出所有文件 ll更详细
touch file 新建文件
mv file file 重命名
mkdir 创建目录
pwd 打印工作目录
cd .. 回到上一级目录
rm file 删除文件 
rm - r dir 移除目录
wc file 统计文件字符数
cat file 查看文件内容
less file 查看文件按Enter逐行显示
cp file dir 复制文件
